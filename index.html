📖 Guide d'utilisation
📥 Import
Glissez ou cliquez pour charger votre fichier Excel (.xlsx/.xlsm) ou PDF. Le planning est extrait automatiquement depuis les onglets Planning ou Arélia.
✅ Fiche Validation
Tableau gauche = postes planifiés à cocher. Tableau droit = postes validés (pointage réel). Utilisez les filtres Agent / Site / Date pour affiner la vue.
👆 Clic sur ligne droite
Un menu contextuel apparaît sous la ligne avec 3 actions : ✏️ Modifier (heures réelles, statut, motif) · ⇄ Échange (désigner un remplaçant) · 🗑 Supprimer (retirer la validation).
➕ Poste Suppl.
Ajoute un poste non planifié (commande client urgente). Agents et sites proposés en autocomplétion depuis le planning importé.
📊 Export Excel
Génère un fichier .xlsx avec deux onglets : NOTIFICATIONS (postes validés avec statuts) et RELEVE DYNAMIQUE (planning complet avec coches).
🗑 Reset
Efface tout le planning, les validations et les extras du navigateur. Une confirmation est demandée. Irréversible.
🔄 Sync Supabase
POINTAGE = envoie les validations cochées (relevé mensuel). PLANNING = envoie le planning source brut importé. Les deux peuvent pointer vers la même base ou des bases différentes.
Statuts de pointage :
Conforme = présent aux horaires prévus  ·  Retard = arrivée tardive  ·  Échange = remplacé par un autre agent  ·  Absent = non présenté
⚙️ Configuration
🔗 Connexion Supabase — KHEOPS POINTAGE
Synchronise les validations de pointage (relevé mensuel coché).
📅 Connexion Supabase — KHEOPS PLANNING
Synchronise le planning importé (postes planifiés source). Peut être la même base ou une base séparée.
